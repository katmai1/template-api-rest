# Template Django

## Basic configuration
### Configure PostgreSQL

Install dependencies:
- `sudo apt-get install postgresql postgresql-contrib`

Create database:
- `./create_database.sh`

Input data in config.env

### Create admin user

- `python manage.py createsuperuser`
